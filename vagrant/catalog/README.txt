Catalog

Steps To Run

1. Open the terminal.
2. Change directories to the vagrant folder.
3. Type ‘vagrant up’.
4. Type ‘vagrant ssh’.
5. Change directories to the catalog folder.
6. If you would like to start a new database, put the catalog.db in the trash and type ‘python database_setup.py’.
7. Type ‘project.py’.
8. Open the browser and type ‘http://localhost:8080/catalog'.
9. Sign in using Google + to add, delete, and edit your own entries.
